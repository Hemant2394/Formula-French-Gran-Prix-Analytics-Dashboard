# 🏎️ Formula 1 French Grand Prix 2021 Analytics Dashboard

An end-to-end data analytics project analyzing the **2021 Formula 1 French Grand Prix** using **Python (Pandas), MySQL, and Power BI**.

## 📌 Project Overview

This project transforms raw Formula 1 race data into an interactive Power BI dashboard.

**Workflow:**
- 🐍 Cleaned and transformed raw data using **Python (Pandas)**
- 🗄️ Imported data into **MySQL** and performed SQL analysis
- 📊 Built an interactive **Power BI Dashboard** to visualize race performance
